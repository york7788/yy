这是一个可以通过企业微信群机器人给群里发送消息的网页程序
可以方便企业同时给多个群一次性推送消息（前提是机器人添加到群）
怎么创建企业群机器人自己研究！

消息类型：
1.文本消息
2.Markdown消息
3.图片消息
4.图片链接消息（图片会上传到服务器）

可以设置立即发送和定时发送！
